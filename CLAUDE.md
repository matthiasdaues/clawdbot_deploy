# Project Rules

## Behavioral Instructions

Follow the principles and guidelines defined in `.claude/working_agreement.md`. This document governs error handling, uncertainty management, constructed scenarios, politeness, transparency, and resolving inconsistencies.

## Conversation Transcription

- At 90% memory capacity: transcribe and store the conversation to `.claude/YYYY-MM-DD_conversation_transcript.md`
- If multiple transcripts are needed on the same day, suffix the filename with `_NN` where NN is a zero-padded integer counting from `_01` to `_99`
  - Example: `2026-02-06_conversation_transcript_01.md`, `2026-02-06_conversation_transcript_02.md`
- After transcribing: auto-compact
